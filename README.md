## 🎯 这是什么

**弥（Metheus）** 是一个专为独立开发者设计的本地桌面应用。它不满足于和单个 AI 聊天，而是帮你**编排一个 AI 专家团队**——按固定流程协同工作，从想法到代码，一步不落。

你负责**说 Yes/No**，AI 团队负责**干活**。

---

## 🔥 它解决了什么痛点

### 现状：你用三个工具，还要自己当胶水

| 工具 | 能干啥 | 缺啥 |
|------|--------|------|
| **Open WebUI** | 多角色群聊协作 | 过不了浏览器沙盒，碰不到本地文件 |
| **Cherry Studio** | 角色管理好 | 流程分散，需要手动复制粘贴 |
| **Claude Code** | 本地执行强，能读写文件 | 没有决策层，要自己写提示词 |

**结果：** 你在三个工具之间不断"复制粘贴、搬运上下文、重复下达指令"。搬运本身浪费的时间，比 AI 干活的时间还多。

### 弥的解法：把"决策"和"执行"分离

```
你的任务不应该是"操作多个AI工具"，
你的任务应该是"在关键节点说 Yes/No"。
```

弥用一个统一的工作流把**决策层**（讨论方案）和**执行层**（写代码）自动接起来。AI 团队各司其职，你只在关键节点拍板。

---

## 💎 核心理念

### 用便宜的决策，换昂贵的执行

独立的开发者最心疼的不是时间——是 **Token 账单**。

| 环节 | Token 消耗 | 出错的代价 |
|------|-----------|-----------|
| 决策层讨论（AI 聊天） | 低（纯文本，几百 token） | 发现方向错了 → 重新讨论，零浪费 |
| 执行层（Claude Code 写代码） | 高（代码生成 + 文件操作） | 跑错方向 → 之前全白费 |

**弥的策略：** 在决策层多花 100 token 把方向对齐，省掉执行层跑偏浪费的 10,000 token。**让每一个 Token 都花在刀刃上。**

```
用便宜的模型做出贵模型的交付效果 = 极致的降本增效
```

## 🏗️ 四层 AI 专家团队

弥不是"一个 AI 陪你聊天"，而是**一个完整的 AI 部门**：

| 角色 | 职责 | 输出 |
|------|------|------|
| 🎯 **策略产品经理** | 和你讨论想法，明确方向、竞品、商业模式 | 版本方案（v1.0 目标蓝图） |
| 📋 **产品经理** | 把方案拆成可交付的小版本 | 大阶段列表（v0.1, v0.2...） |
| 🔧 **域负责人** | 技术落地分割，每个模块都是垂直切片 | 中阶段列表（v0.1.1, v0.1.2...） |
| ⚡ **开发工程师** | 把任务拆成 Claude Code 可执行的原子步骤 | 精确提示词 + 自动执行 |

**分层对话，不是一问四答。** 当前进展到哪一步，只有对应角色说话。需要谁，@ 一下就能召唤过来。

## 🔄 核心工作流

```
你的一个想法
    ↓
和策略产品经理讨论、辩论、打磨
    ↓
方向定了 → 生成版本方案 → 你点 [✅ 批准]
    ↓
产品经理自动拆成可交付的版本（v0.1, v0.2, v0.3...）
    ↓
域负责人拆成技术实现模块（v0.1.1, v0.1.2...）
    ↓
开发工程师动态生成 Claude Code 提示词
    ↓
Claude Code 在本地项目中读写文件，真实生成代码
    ↓
测试工程师每个小阶段自动检查代码质量
    ↓
每个版本完成后 → 你审阅报告 → [✅ 批准] / [❌ 驳回]
    ↓
进入下一版本迭代
```

**灵魂设计：** 所有关键产出（方案、版本、代码）都必须经过你的一键审批——**你不是被 AI 取代的人，你是指挥 AI 团队的人。**

## ⚡ 两种模式，适配所有项目

| | 🚀 快速模式 | 🏗️ 专业模式 |
|------|-----------|-----------|
| 层级 | 2 级（大阶段 → 小阶段） | 3 级（大阶段 → 中阶段 → 小阶段） |
| 适用 | 小想法、原型验证、个人工具 | 大型项目、企业级应用 |
| 审批 | 大阶段完成后审批 | 每个中阶段完成后审批 |
| 节奏 | 快，一步到位 | 稳，层层把关 |

**随时切换。** 同一项目里，简单的版本用快速模式，复杂的版本用专业模式。

## 🖥️ 技术特性

- **本地优先：** 桌面应用（非 Web），代码和 API Key 都在你的机器上
- **无缝文件访问：** AI 直接读写你的项目文件，不需要手动上传下载
- **多模型兼容：** 当前支持 DeepSeek + Claude，后续扩展更多
- **轻量高效：** Tauri 打包（非 Electron），内存占用低
- **持久化：** 项目自动保存到本地，关掉重开不丢失

| 层 | 技术 |
|------|------|
| 前端 UI | React + TypeScript |
| 桌面容器 | Tauri 2 |
| 后端服务 | Rust |
| AI 执行层 | Claude Code CLI + DeepSeek API |

## 🚀 快速开始

### 环境要求

- Node.js ≥ 18
- Rust ≥ 1.70
- Ubuntu / macOS / Windows
- Claude Code CLI（可选，用于真实代码执行）

### 安装与启动

```bash
# 克隆项目
git clone https://github.com/nicepkg/metheus.git
cd metheus

# 安装前端依赖
npm install

# 启动开发模式
npm run tauri dev
```

### 配置 API Key

在项目根目录创建 `.env` 文件：

```env
DEEPSEEK_API_KEY=你的DeepSeek密钥
ANTHROPIC_API_KEY=你的Anthropic密钥
```
